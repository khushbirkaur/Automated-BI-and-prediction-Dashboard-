# 📊 Automated Business Intelligence & Sales Prediction Dashboard

## 🚀 Project Overview

The **Automated Business Intelligence & Sales Prediction Dashboard** is an end-to-end data analytics platform built using Python and Streamlit. The system automatically processes sales data, generates interactive visualizations, and predicts future sales using machine learning.

This project demonstrates how **Business Intelligence, Data Visualization, and Machine Learning** can be integrated into a single cloud-based analytics platform.

The dashboard provides insights into sales performance across regions, states, categories, and customers while also forecasting future revenue trends.

# 📌 Features

### 📈 Automated BI Dashboard

* Real-time KPI metrics
* Sales trend analysis
* Regional performance insights
* Category-wise sales distribution
* Shipping mode analysis

### 🎛 Interactive Filters

Users can dynamically filter the dashboard by:

* Year
* Region
* State
* City
* Category
* Customer Segment

### 📊 KPI Cards

Key performance indicators displayed:

* Total Sales
* Total Orders
* Average Sales
* Average Shipping Time
* Unique Customers
* Total States
* Total Cities
* Total Products

### 🔎 Advanced Insights

* Top 10 customers by revenue
* Top 10 products by sales
* Top performing states

### 🤖 Machine Learning Prediction

The dashboard uses **Linear Regression** to forecast future sales for upcoming years based on historical sales data.

### ☁ Cloud Deployment

The application is deployed using Streamlit Cloud and integrates with Google Sheets for real-time data updates.

# 📊 Data Source

The dataset used in this project contains retail sales information including:

* Order Date
* Ship Date
* Customer Details
* Region / State / City
* Product Category
* Sales

The data is stored in **Google Sheets**, enabling the dashboard to update automatically whenever the dataset changes.
